\item[Title]\AsIs{Differentially Private Scree and Score Visualization}
\item[Version]\AsIs{0.1.0}
\item[Description]\AsIs{Tools for performing principal component analysis (PCA) and
constructing differentially private scree estimates and score-based
visualizations. The package provides non-private and private PCA directions,
DP scree estimators based on clipped means, Huber-type private means, and
private winsorized means, as well as 2D score histograms with additive and
sparse DP mechanisms, including group-wise visualizations.}
