# Five Gaussian clusters with group labels

A grouped version of
[`gau`](https://yejinjo0220.github.io/dppca/reference/gau.md) with an
additional group label column.

## Usage

``` r
gau_g
```

## Format

A data frame with 5,000 rows and 21 columns. The first 20 columns, named
`V1`, `V2`, ..., `V20`, are the numerical variables from
[`gau`](https://yejinjo0220.github.io/dppca/reference/gau.md). The last
column, `group`, is a group label with values `group1`, `group2`,
`group3`, `group4`, and `group5`.

## Source

Generated by the authors of the package.

## Details

The dataset contains the same 5,000 observations and 20 numerical
variables as
[`gau`](https://yejinjo0220.github.io/dppca/reference/gau.md), together
with a group label column. There are five groups, and each group
contains 1,000 observations.
