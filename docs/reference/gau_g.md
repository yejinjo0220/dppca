# Five Gaussian clusters with group labels

A grouped version of
[`gau`](https://yejinjo0220.github.io/dppca/reference/gau.md) with an
additional group label column.

## Usage

``` r
gau_g
```

## Source

Simulated by the authors of the package. The data-generating code is
available in `data-raw/make_datasets.R` in the package source
repository.

## Details

The dataset contains 5,000 observations in 20 dimensions and one group
label column. There are five groups, and each group contains 1,000
observations.
