# Articles

### Theory

- [Principal Component Analysis
  Background](https://yejinjo0220.github.io/dppca/articles/pca_background.md):

  Background notation for population PCA, sample PCA, principal
  component directions, scores, scree values, and explained variance
  ratios.

- [PC Directions in
  dppca](https://yejinjo0220.github.io/dppca/articles/pc_direction.md):

  Principal component direction estimation used in dppca, including
  non-private sample PCA directions and differentially private g-DPPCA
  directions.

- [DP scree in
  dppca](https://yejinjo0220.github.io/dppca/articles/dp_scree.md):

  Differentially private PCA scree estimation in dppca including clipped
  mean, Huber GDP mean, and private modified winsorized mean methods.

- [DP Score plot in
  dppca](https://yejinjo0220.github.io/dppca/articles/dp_score.md):

  Differentially private PCA score visualization in dppca, including
  private plotting frames, bin selection, additive DP histograms, sparse
  DP histograms, and group-wise score histograms.
